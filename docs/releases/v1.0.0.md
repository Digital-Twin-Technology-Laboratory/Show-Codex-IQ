# Codex Toolbox v1.0.0

Show Codex IQ 正式升级为 Codex Toolbox。原有 Codex 模型智商、费用、耗时、综合排名与菜单栏功能全部保留，新增本机 Token 用量与账户重置卡只读查询。

## 新功能

- 今日本机原始 Token 总量、根任务 Top 3 与 7/14/30/90 天趋势；榜单点击后可按设置展开为 Top 5 或 Top 10，任务不足时显示实际项数，趋势柱悬停显示精确用量。
- 每项 Token 后显示对账户 5 小时/周窗口的估算占比；按账户已用比例与本机同期 Token 推算，仅供参考。
- 模型趋势图默认关闭，可开启并选择 7/14/30/90 天范围。
- 增量本地账本，支持历史回填、重复累计事件去重、损坏/缺失源标记与手动清除。
- 重置卡可用数量、最近过期时间和脱敏详情。
- 可隐藏、折叠和排序的三模块看板，以及按功能分组的完整设置。
- Token 与重置卡默认折叠，折叠标题仍显示今日 Token 总量与可用卡数。
- “通用&看板”设置整合模块排序、登录启动与 GitHub 正式 Release 自动更新检查。
- 模块标题整行均可展开/折叠，并修复了 macOS 27 弹窗尺寸动画重入引发的闪退。

## 升级与隐私

- Bundle ID 不变，继承原设置、排名缓存与登录启动偏好。
- PKG 安全迁移 `/Applications/Show Codex IQ.app`，新应用验证失败时不删除旧副本。
- Token 审计不调用模型、不上传内容。
- 重置卡只读，不保存 opaque ID，不提供兑换、删除或自动使用。
- 重置卡逐卡只展示发放时间和过期时间（北京时间），不输出 access token、refresh token、cookie、说明文字或完整唯一 ID。

## 安装包

同一 Release 提供 `Codex-Toolbox-1.0.0-universal.pkg` 和 `Codex-Toolbox-1.0.0-universal.dmg`，以及各自 `.sha256`。PKG 是从 Show Codex IQ 升级的推荐方式；使用 DMG 升级时，请先退出并删除 `/Applications/Show Codex IQ.app`，再拖入新应用，避免新旧并存。

v1.0.0 已于 **2026 年 7 月 22 日**正式发布。PKG 与 DMG 均使用 Developer ID 签名，通过 Apple 公证并已 staple ticket；两个安装包均为 Apple Silicon 与 Intel 通用的 Universal 2 产物。
